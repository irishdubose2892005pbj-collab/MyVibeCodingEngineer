## 📝 问题记录规范

每个 Debug 记录应使用 [Debug模板](../99-Templates/Debug模板.md)，包含：

1. **问题描述**: 现象、错误信息、发生环境
2. **复现步骤**: 如何稳定复现这个问题
3. **原因分析**: 根因定位过程
4. **解决方案**: 具体的解决步骤
5. **预防措施**: 如何避免再次踩坑
6. **相关链接**: Stack Overflow、GitHub Issue、文档

---

## 💡 记录原则

- **即时记录**: 解决问题后 10 分钟内完成记录
- ** searchable**: 标题和标签要包含关键词
- **归因清晰**: 区分根本原因和表面现象
- **持续积累**: 定期回顾，提取通用模式
