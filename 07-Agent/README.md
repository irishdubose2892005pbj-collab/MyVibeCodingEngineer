## 📝 设计原则

1. **单一职责**: 每个 Agent/工具只做一件事，做好一件事
2. **可观测性**: 每个决策步骤都可追踪、可调试
3. **容错设计**: 工具失败时有降级方案
4. **成本意识**: Token 消耗和延迟是设计约束
5. **人机协作**: 明确 Agent 和人类的职责边界
