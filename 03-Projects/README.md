## 🏗️ 新项目启动流程

1. **复制模板**: 从 [99-Templates/项目模板.md](../99-Templates/项目模板.md) 创建项目 README
2. **创建目录**: 在 `03-Projects/` 下新建项目文件夹
3. **定义目标**: 明确项目的核心功能和成功标准
4. **技术选型**: 选择合适的技术栈
5. **开发迭代**: 使用 Vibe Coding 方式快速推进
6. **部署上线**: 记录部署过程和访问地址
7. **项目复盘**: 使用 [项目复盘模板](../99-Templates/项目复盘模板.md) 总结

## 📊 项目状态定义

- `⬜ 未开始` — 已规划，待启动
- `🔄 进行中` — 正在开发
- `✅ 已完成` — 功能完成，已部署
- `⏸️ 暂停` — 暂时搁置
- `🗃️ 归档` — 不再维护，仅留档
