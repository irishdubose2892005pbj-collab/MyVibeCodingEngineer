## 📝 Prompt 记录规范

每个 Prompt 文件应包含：

1. **用途说明**: 这个 Prompt 解决什么问题
2. **适用模型**: 推荐使用的 AI 模型
3. **输入变量**: 需要替换的占位符
4. **Prompt 正文**: 完整的提示词
5. **输出示例**: 预期的输出样例
6. **效果评分**: 1-5 星，方便筛选

**模板**: 参考 [Prompt模板](../99-Templates/Prompt模板.md)

## ⭐ 精品 Prompt 标准

- 输出稳定，多次运行结果一致
- 无需过多后续修改即可直接使用
- 有明确的输入输出边界
- 经过实际项目验证
