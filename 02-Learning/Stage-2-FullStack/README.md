# Stage-2: FullStack 全栈开发

> 从原型到产品：掌握构建完整 Web 应用的技术栈

**预计耗时**: 4-8 周  
**产出目标**: 1 个完整的全栈应用（前端 + 后端 + 数据库 + 部署）

---

## 🎯 阶段目标

1. 掌握现代前端框架（React / Next.js）
2. 理解后端服务开发（API 设计、认证、业务逻辑）
3. 学会数据库设计与操作
4. 掌握应用部署与运维基础

---

## 📋 学习任务清单

### Frontend
- [ ] HTML/CSS/JS 基础强化
- [ ] React 核心概念：组件、Hooks、状态管理
- [ ] Next.js：SSR、路由、API Routes
- [ ] UI 框架：Tailwind CSS / shadcn-ui / Ant Design
- [ ] **实践**: 用 Next.js 搭建一个博客/管理后台

### Backend
- [ ] Node.js / Express 或 Python / FastAPI
- [ ] RESTful API 设计规范
- [ ] 用户认证：JWT / Session / OAuth
- [ ] 中间件、错误处理、日志记录
- [ ] **实践**: 为前端项目开发配套 API 服务

### Database
- [ ] SQL 基础与 PostgreSQL / MySQL
- [ ] NoSQL 概念与 MongoDB / Redis
- [ ] ORM 使用：Prisma / TypeORM / SQLAlchemy
- [ ] 数据库设计：ER 图、索引、迁移
- [ ] **实践**: 设计并实现项目的数据库 schema

### API
- [ ] REST API 设计最佳实践
- [ ] API 文档：Swagger / OpenAPI
- [ ] 版本控制与向后兼容
- [ ] 速率限制与安全防护
- [ ] **实践**: 为项目编写完整的 API 文档

### Deployment
- [ ] Docker 容器化基础
- [ ] 云平台：Vercel / Railway / 阿里云
- [ ] CI/CD 基础：GitHub Actions
- [ ] 域名、HTTPS、监控
- [ ] **实践**: 将全栈应用部署上线
