## 📖 各阶段详解

### [Stage-0: 基础环境](./Stage-0-基础环境/README.md)
**目标**: 搭建高效的 AI 开发工作流

- [ ] Git 基础与 GitHub 工作流
- [ ] 开发环境配置（Cursor / VSCode / Claude Code）
- [ ] Markdown 与文档规范
- [ ] AI 工具链认知（ChatGPT / Claude / Kimi）
- [ ] 本知识库的使用规范

### [Stage-1: Vibe Coding](./Stage-1-Vibe-Coding/README.md)
**目标**: 掌握 AI 辅助编程，快速产出可运行的产品原型

- [ ] 贪吃蛇游戏（HTML + JS，熟悉 AI 编程范式）
- [ ] AI 能力边界探索（什么能做，什么不能做）
- [ ] Vibe Coding 方法论（Prompt 工程、迭代调试）
- [ ] Dify 平台使用（低代码 AI 应用搭建）
- [ ] AI 产品原型设计
- [ ] 项目实践

### [Stage-2: FullStack](./Stage-2-FullStack/README.md)
**目标**: 掌握全栈开发，能独立构建完整 Web 应用

- [ ] Frontend: React / Vue / Next.js
- [ ] Backend: Node.js / Python / Go
- [ ] Database: SQL / NoSQL / ORM
- [ ] API: REST / GraphQL / WebSocket
- [ ] Deployment: Docker / Vercel / 阿里云

### [Stage-3: Agent](./Stage-3-Agent/README.md)
**目标**: 理解并构建智能体系统

- [ ] Workflow 编排
- [ ] Memory 记忆系统
- [ ] MCP 协议
- [ ] Multi-Agent 协作
- [ ] Harness 评测与监控

## 🛠️ 学习原则

1. **项目驱动**: 每个阶段至少产出一个可运行的项目
2. **文档先行**: 学习前先写预期，学习后写总结
3. **问题即资产**: 遇到的每个 Bug 都是 `06-Debug/` 的素材
4. **Prompt 即代码**: 有价值的 Prompt 及时归档到 `05-Prompts/`
5. **输出倒逼输入**: 学完一个知识点，尝试写一篇教程或做一个 Demo
