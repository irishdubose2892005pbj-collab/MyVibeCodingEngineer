## 🚀 快速开始

### 今日工作流
1. 在 `01-Daily/2026/` 创建/打开今日日志（使用 [Daily模板](../99-Templates/Daily模板.md)）
2. 从 `02-Learning/` 当前阶段选取今日学习任务
3. 学习过程中的问题记录到 `06-Debug/`
4. 有价值的 Prompt 保存到 `05-Prompts/`
5. 项目进展更新到 `03-Projects/`

### 学习路径总览
```
Stage-0: 基础环境 Setup
    ↓
Stage-1: Vibe Coding (AI 辅助编程)
    ↓
Stage-2: FullStack (全栈技术栈)
    ↓
Stage-3: Agent (智能体系统)
```

## 📝 维护规则

- **Daily**: 每个工作日创建一篇日志
- **Weekly**: 每周日写周复盘
- **Projects**: 每个项目一个独立目录，使用 [项目模板](../99-Templates/项目模板.md)
- **Knowledge**: 采用原子化笔记，一概念一文件
- **Templates**: 更新模板时同步更新此索引
