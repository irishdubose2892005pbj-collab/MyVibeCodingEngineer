# 📦 资源资产库

> 项目相关的媒体资源和临时文件

---

## 📂 目录说明

| 目录 | 用途 | 管理规范 |
|------|------|----------|
| [Images](./Images/) | 项目图片、Logo、图标 | 按项目分子目录，命名清晰 |
| [Screenshots](./Screenshots/) | 截图、录屏帧 | 按日期+项目命名 |
| [PDFs](./PDFs/) | 文档、报告、论文 | 按来源和主题分类 |
| [Videos](./Videos/) | 视频素材、Demo | 压缩后上传，避免大文件 |
| [Audio](./Audio/) | 音频素材、录音 | 标注用途和时长 |
| [Temp](./Temp/) | 临时文件 | 定期清理，不长期保留 |

---

## ⚠️ 使用规范

1. **命名规范**: `YYYYMMDD-项目名-描述.扩展名`
2. **大小控制**: Git 不跟踪大文件，>10MB 使用外部存储
3. **版权意识**: 商用素材确保有授权
4. **定期清理**: Temp 目录每月清理一次
